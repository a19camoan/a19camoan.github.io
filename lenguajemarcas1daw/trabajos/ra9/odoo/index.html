Instalación y configuración Odoo

Instalación 1

Al descargar y abrir el ejecutable lo primero que nos pedirá es el idioma (solo están disponibles inglés y francés) y podremos seguir la Instalación con el idioma preferido.

Instalación 2

En este paso nos pedirá qué tipo de instalación queremos ya que podemos instalar el servidor de odoo, la base de datos PostgreSQL o ambos.

Instalación 3

Nos pedirá el nombre y contraseña de usuario junto con el hostname y el puerto (puede ser editado posteriormente).

Configuración 1

Al abrir Odoo esto será lo primero que veamos. Rellenaremos los campos para terminar de añadir los datos necesarios para crear la base de datos.

Configuración 2

Y con esto ya tendríamos instalado y configurado Odoo.
